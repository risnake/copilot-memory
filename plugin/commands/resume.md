# Resume Command

Resume from the latest handoff note.

## Usage

```bash
copilot-memory resume
```

## What it does

- Finds the most recent handoff note
- Displays its content for context
- Helps you pick up where you left off

## Example

```bash
$ copilot-memory resume
Found latest handoff: Sprint 1 Complete (2024-01-15)

## Completed
- User authentication
- Database setup

## Next Steps
- API endpoints
- Testing suite
```

Use this at the start of each work session to quickly restore context.
