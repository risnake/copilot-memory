# Help Command

Show command help and usage information.

## Usage

```bash
# List all commands
copilot-memory help

# Get help for specific command
copilot-memory help <command>
```

## Examples

```bash
# Show all commands
copilot-memory help

# Get handoff command help
copilot-memory help handoff

# Get phase commands help
copilot-memory help phase

# Get vault commands help
copilot-memory help vault
```

## Command Groups

**Core Commands:**
- init, handoff, resume, help

**Phase Commands:**
- phase create, phase research, phase handoff

**Vault Commands:**
- vault index, vault search, vault doctor, vault prune, vault tracker

Use `copilot-memory help <command>` for detailed usage.
